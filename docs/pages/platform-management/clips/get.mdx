`GET https://app.resemble.ai/api/v2/projects/{project_uuid}/clips/{clip_uuid}`

## Example Request

```bash
curl 'https://app.resemble.ai/api/v2/projects/PROJECT_UUID/clips/CLIP_UUID' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

`GET https://app.resemble.ai/api/v2/voice_settings_presets/{uuid}`

Retrieve a specific voice settings preset.

| Parameter | Type | Required | Notes |
| --- | --- | --- | --- |
| `uuid` | string | ✅ | Preset UUID (URL parameter). |

```bash
curl 'https://app.resemble.ai/api/v2/voice_settings_presets/PRESET_UUID' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

`POST https://app.resemble.ai/api/v2/projects`

| Field | Type | Required | Notes |
| --- | --- | --- | --- |
| `name` | string | ✅ | Project name (max 256 characters). |
| `description` | string | ❌ | Project description (max 1024 characters). |
| `is_collaborative` | boolean | ❌ | Whether project is collaborative. |
| `is_archived` | boolean | ❌ | Whether project is archived. |

```bash
curl --request POST 'https://app.resemble.ai/api/v2/projects' \
  -H 'Authorization: Bearer YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  --data '{
    "name": "Podcast Season 2",
    "description": "Episodes and promotional clips",
    "is_collaborative": true,
    "is_archived": false
  }'
```

```json
{
  "success": true,
  "item": {
    "uuid": "PROJECT_UUID",
    "name": "Podcast Season 2",
    "description": "Episodes and promotional clips",
    "is_collaborative": true,
    "is_archived": false,
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
}
```

`GET https://app.resemble.ai/api/v2/pronunciations`

Retrieve all custom pronunciations for your team, with optional filters.

### Query Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `language` | string | Filter by language (e.g., `en-us`). |
| `domain` | string | Filter by domain (e.g., `medical`). |
| `status` | string | Filter by status: `pending`, `ready`, or `failed`. |
| `active` | boolean | Filter by active state. |

```bash
curl "https://app.resemble.ai/api/v2/pronunciations?status=ready" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

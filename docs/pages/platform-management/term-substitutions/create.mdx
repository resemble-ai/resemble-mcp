---
title: Create Term Substitution
slug: /platform-management/term-substitutions/create
---

`POST https://app.resemble.ai/api/v2/term_substitutions`

```json
{
  "original_text": "Vodaphone",
  "replacement_text": "voduhfone"
}
```

```bash
curl --request POST 'https://app.resemble.ai/api/v2/term_substitutions' \
  -H 'Authorization: Bearer YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  --data '{
    "original_text": "Vodaphone",
    "replacement_text": "voduhfone"
  }'
```

Response includes the created substitution with timestamps.

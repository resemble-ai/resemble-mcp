`DELETE https://app.resemble.ai/api/v2/term_substitutions/{substitution_uuid}`

```bash
curl --request DELETE 'https://app.resemble.ai/api/v2/term_substitutions/SUB_UUID' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

Response: `{ "success": true }`

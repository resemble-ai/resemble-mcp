title: List Term Substitutions
slug: /platform-management/term-substitutions/list
---

`GET https://app.resemble.ai/api/v2/term_substitutions`

| Parameter | Type | Notes |
| --- | --- | --- |
| `page` | number ≥ 1 | Defaults to 1. |
| `page_size` | number 10–1000 | Defaults to 10. |

```bash
curl 'https://app.resemble.ai/api/v2/term_substitutions?page=1&page_size=10' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

```json
{
  "success": true,
  "page": 1,
  "num_pages": 2,
  "page_size": 10,
  "items": [
    {
      "uuid": "SUB_UUID",
      "original_text": "Vodaphone",
      "replacement_text": "voduhfone",
      "created_at": "2024-01-15T10:30:00Z",
      "updated_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```

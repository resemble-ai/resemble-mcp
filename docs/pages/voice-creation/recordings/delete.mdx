`DELETE https://app.resemble.ai/api/v2/voices/{voice_uuid}/recordings/{uuid}`

Delete a recording.

| Parameter | Type | Required | Notes |
| --- | --- | --- | --- |
| `voice_uuid` | string | ✅ | Voice UUID (URL parameter). |
| `uuid` | string | ✅ | Recording UUID (URL parameter). |

```bash
curl --request DELETE 'https://app.resemble.ai/api/v2/voices/VOICE_UUID/recordings/RECORDING_UUID' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

```json
{
  "success": true
}
```

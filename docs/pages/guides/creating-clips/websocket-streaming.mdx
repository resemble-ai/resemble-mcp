> **Business plan required:** The WebSocket API is available to Business-tier accounts. Upgrade on the [billing page](https://app.resemble.ai/account/billing) if you receive authorization errors.

Stream audio as it is generated to deliver low-latency playback for assistants, live dubbing, and interactive demos.

## Quick Setup

```bash
pip install websockets pyaudio python-dotenv
```

Create a `.env` file with `RESEMBLE_API_KEY=your_token` and pick a streaming-enabled `voice_uuid`.
